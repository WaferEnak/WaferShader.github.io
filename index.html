    • Wafer Shader PE

    A vibrant shader with simple features made especially for Minecraft PE

    What Is Wafer Shader?

    Wafer Shader is a vibrant themed shades, with simple but attractive features.

    You can change anything in this shader, because this shader has quite a lot of customization but not too much, this shader also has parallax features (but still in the experimental stage and there may still be bugs), volume cloud and square cloud, water bumps, and etc...

    You can customize the shaders, by changing the value in the shader file located at(WaferShader > shaders > glsl > lib > Adj.h), you can also select the graphic set you want(Quality, Balance, Performance) via subpacks.

    This shader is not completely perfect and optimal, maybe there is a bug that I don't know about, you can report to me if there is a bug.


    How to instal?

  • First you have to download this shader first
  • After you download the shader, you have to extract it
  • After you extract it, move it to file android > data > com.mojang.minecraftpe > files > games > com.mojang > resource_packs, move it to that resource_packs file
  • after moving, open minecraft pe then go to settings > global resource packs > ha? > choose WaferShader
  • And congratulations!!! you have been installed Wafer Shader
